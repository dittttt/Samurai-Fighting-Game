# WAGO Complete Game Roadmap

WAGO is a story-driven pixel-art samurai game set on Froy. The original Java prototype is a 1v1 fight against a bot; the active direction is now a Godot 4.x port inspired by Katana ZERO, Blasphemous, and Hyper Light Drifter.

## Game direction

Side-scrolling samurai action with:

- precise sword combat
- parry/guard/roll
- dark political story about Elk and the Gabals
- tournament arc in Pahica
- Lyon archive/lore sections
- Umbral Folk and Bhunt mechanic
- Arthur mentor/boss arc

See `docs/GODOT_PORT_PLAN.md` for implementation steps.
