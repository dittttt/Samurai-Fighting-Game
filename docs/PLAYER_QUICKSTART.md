# WAGO Player Quickstart

You do not need to understand Godot to run the current prototype.

## Easiest way

Double-click this file in the repository root:

```text
RUN_WAGO.bat
```

It uses your local Godot install:

```text
C:\Softwares\Godot_v4.7-stable_win64.exe
```

## If you are already in Godot

1. Open/import:

```text
C:\Users\ditto\Desktop\CODES\SIDE PROJECTS\JAVA GAMES\Samurai-Fighting-Game\godot\project.godot
```

2. Press the **Play** button in the top-right.
3. If Godot asks for the main scene, choose:

```text
res://scenes/main/Main.tscn
```

## Controls

| Action | Key |
| --- | --- |
| Move | A/D or Left/Right |
| Jump | Space, W, or Up |
| Run | Shift |
| Roll / dodge | C or Ctrl |
| Light attack | J |
| Heavy attack | K |
| Pause | P or Esc |
| Restart fight | R |
